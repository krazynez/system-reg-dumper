# Setup
* Copy and merge the `PSP` dir to the root of your memory stick `ms0:/`
* or copy SYSTEM_REG_DUMPER folder to `ms0:/PSP/GAME/`

# Prerequisites 
* You need to run DAM ( DCARK Maker ) before hand. 
* If you used DCARK Installer via PSP app you do not need to run this at all as it already copies the files it needs.

# Run
* Press X and it will copy the system.dreg and system.ireg to the proper location ( ms0:/TM/DCARK/registry )

* because I could not get DCARK to copy from flash1 when it initially boots yet so here is once alternative.



# Fun fact
* You can just choose the `Boot ARK from Memory Stick` and once you do the initial setup it will also create these files.
